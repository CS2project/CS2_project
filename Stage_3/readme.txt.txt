How to Compile and Run the Program

Prerequisites
Make sure you have Java JDK installed on your system.
You should have all the .java files in the same directory.

Step 1: Open the Terminal
Navigate to the directory where your project files are located using the terminal/command prompt.

Step 2: Compile the Java Files
javac *.java
This command compiles all the .java files in the current directory.

Step 3: Run the Program
java BanderaCafeSystem
This command runs the main program file, starting the application.
